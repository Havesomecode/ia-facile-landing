# Reassuring Editorial

This direction treats IA Facile like a calm editorial guide for adults who feel slightly behind or intimidated by AI.

Visual thesis:

- typographic hierarchy first
- spacious editorial bands
- restrained warmth
- minimal containers
- clarity before decoration

What it should feel like:

- adult
- literate
- calm
- respectful
- unhurried

What it should avoid:

- centered hero inside a rounded box
- stacked SaaS cards
- glossy surfaces
- pill-heavy component rhythm
- startup purple gradients
