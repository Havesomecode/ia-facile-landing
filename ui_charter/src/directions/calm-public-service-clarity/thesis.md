# Calm Public-Service Clarity

This direction treats IA Facile like a modern public-interest guide.

The page should feel:

- oriented
- reassuring
- stable
- sober
- useful on first read

The structure should prioritize:

- explicit hierarchy
- clear signposting
- flat surfaces
- disciplined borders
- minimal visual noise

The direction should avoid:

- nested rounded cards
- ornamental composition
- vague premium emptiness
- startup gradients
- component repetition as identity
