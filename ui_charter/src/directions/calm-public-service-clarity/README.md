# Calm Public-Service Clarity

Visual thesis:

- modern public-interest guide
- explicit hierarchy and signposting
- flat surfaces with disciplined borders
- sober, welcoming, highly legible typography
- restrained accent colors

What this direction avoids:

- nested rounded cards
- decorative layering
- vague premium-minimal emptiness
- soft SaaS panels
- generic startup gradients

What this direction should feel like:

- oriented
- reassured
- stable
- useful
- understandable on first read
