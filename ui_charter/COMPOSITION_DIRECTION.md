# IA Facile Composition Direction

This document fixes the current composition decision for the landing page explorations.

It is not about color, typography, or styling details.
It is only about page architecture.

## Why Composition Matters Here

The landing page is not just trying to look attractive.
It has to do three jobs at once:

- reassure people who feel blocked or intimidated by AI
- make the offer understandable very quickly
- help gauge whether there is real need and which promise resonates

That means the composition should reduce ambiguity.

It should not feel:

- like a generic marketing stack
- like a soft lifestyle brand
- like a SaaS product page

It should feel:

- organized
- legible
- adult
- trustworthy
- specific

## Recommended Composition Principle

Use a **linear, signposted, section-led composition** with selective editorial tension.

In shorthand:

```text
more guided document
less marketing template
```

This means:

- the page should read top to bottom in a very clear order
- each section should have a distinct role
- asymmetry can exist, but only to sharpen hierarchy
- the page should not rely on repeated interchangeable blocks

## Core Structural Logic

The page should answer these questions in order:

1. Is this for me?
2. What exactly is the problem being solved?
3. What first step is being offered?
4. Why trust this approach?
5. What do I do now?

That suggests this composition:

```text
Hero
→ Positioning / reassurance
→ First-step explanation
→ Trust / differentiation
→ CTA
```

This is already the shared story flow in the lab.
The decision here is how those sections should relate spatially.

## Recommended Page Architecture

### 1. Hero: split but disciplined

Use:

- large headline area on the left
- smaller summary / orientation panel on the right

Purpose:

- immediate recognition
- immediate clarity
- visible structure without needing visual tricks

Why:

- this feels more specific and more organized than a centered hero
- it creates confidence without feeling like a dashboard

### 2. Middle sections: signposted bands

Use:

- full-width horizontal bands or rows
- clear section labels
- strong heading + supporting text structure
- occasional two-column behavior where useful

Purpose:

- make each section feel like a step in a guided path
- preserve rhythm without falling into card repetition

Why:

- this matches the need for reassurance and orientation
- it makes the page easier to scan and discuss

### 3. Trust and CTA: heavier anchoring near the end

Use:

- a slightly more grounded final block
- stronger structural closure
- CTA area that feels decisive but not salesy

Purpose:

- turn the page from explanation into action
- create a clean ending rather than a fading tail

Why:

- the page needs to help gauge need, not just create mood

## What To Avoid In Composition

Avoid:

- centered hero + stacked cards underneath
- interchangeable section boxes
- too much symmetry everywhere
- soft floating panels that weaken the reading path
- multiple visual centers competing at once
- page sections that feel decorative rather than functional

## Use Of Asymmetry

Asymmetry is useful, but it should be controlled.

Good use:

- hero split
- occasional offset text / support column
- slight tension between heading block and explanatory block

Bad use:

- collage-like randomness
- decorative imbalance with no informational benefit

The rule:

```text
Use asymmetry to clarify,
not to stylize for its own sake.
```

## Density Decision

The page should feel moderately dense.

Not:

- empty premium minimalism
- crowded product marketing

But:

- enough information to feel serious
- enough spacing to feel calm

This is important because IA Facile needs to feel both:

- accessible
- credible

## Recommended Baseline For Next Iterations

Use `calm-public-service-clarity` as the composition baseline.

Borrow from `reassuring-editorial` only where it improves:

- typographic dignity
- section rhythm
- adult tone

Do not borrow from `warm-modern-guidance` on composition if it reintroduces:

- soft atmospheric spread
- roundness-led structure
- less explicit organization

## Current Decision

If a direction departs from this baseline, it should justify why.

For now, the preferred composition is:

- linear
- section-led
- visibly signposted
- split in the hero
- ruled rather than boxed
- moderately dense
- specific rather than atmospheric
