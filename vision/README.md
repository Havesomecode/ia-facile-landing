# IA Facile Vision

This folder is the working memory for the IA Facile direction.
It is not a final spec and it is not a finished product plan.
It is a structured snapshot of the current thinking, the main choices already made, and the questions that still need validation.

## Start Here

- Read [ABSTRACT.md](./ABSTRACT.md) for the 2-minute overview.
- Read [00-context/why-this-exists.md](./00-context/why-this-exists.md) if you want the framing for the whole exploration.
- Read [00-context/open-questions.md](./00-context/open-questions.md) if you want the unresolved issues first.

## Current Snapshot

The strongest working choices so far are:

- IA Facile is still in exploration and validation mode.
- The landing page is meant to be a market-discovery tool, not just a brochure site.
- The emotional core is reassurance before performance.
- The best current framing is helping non-technical people stop seeing AI as a black box and start using it calmly in everyday life.
- The first audience is more likely hesitant beginners than ambitious early adopters.
- The strongest first offer is a guided first-steps email sequence or short beginner program, paired with light qualification to learn what blocks people.
- The visual direction should feel calm, adult, warm, legible, and non-startupy.
